Descrição do Curso

O curso ‘Dados 360′ traz um panorama completo para quem quer começar a trabalhar com dados na área da comunicação. Você terá acesso a 20 horas de videoaulas e aulas ao vivo gravadas de outras edições, com possibilidade de emissão de certificado. O acesso ao material é imediato, após a confirmação do pagamento.

Ementa do Curso

Semana 1 - Fundamentos e obtenção de dados
1.1 Boas vindas e inspirações 1.2 O que são dados? 1.3 Métodos de obtenção de dados 1.4 Descubra novas fontes de dados 1.5 Introdução a Open Source Intelligence 1.6 Busca avançada 1.7 Privacidade e dados pessoais
Semana 2 — Introdução à programação e banco de dados
2.1 Introdução ao pensamento computacional 2.2 Introdução à programação e automação 2.3 Tecnologias web 2.4 Introdução a bancos de dados 2.5 Introdução ao SQL 2.6 Introdução a R e Python
Semana 3 — Análise de dados
3.1 Tratamentos básicos para dados tabulares 3.2 Cruzamento de dados 3.3 Introdução à estatística descritiva 3.4 Medidas descritivas 3.5 Análise exploratória de dados 3.6 Conceitos das ciências de dados
Semana 4 - Visualização de dados
4.1 Por que “visualização”? 4.2 Elementos na visualização de dados 4.3 Visualização para análise e comunicação 4.4 Outras mídias e formatos 4.5 Canais de percepção 4.6 Usos do espaço 4.7 Usos da cor 4.8 Boas práticas na visualização de dados 4.9 Camadas de uma visualização 4.10 Gráfico comuns (e incomuns) 4.11 Antropográficos 4.12 Ferramentas para visualizar dados
Semana 5 - Trabalhando com textos, mapas e redes sociais
5.1 Texto como dado 5.2 Ferramentas e exemplos com textos 5.3 Análise de redes: conceitos e métricas 5.4 Análise de redes: exemplos e referências 5.5 Introdução aos dados geográficos e mapas 5.6 Caixa de ferramentas para geodados
